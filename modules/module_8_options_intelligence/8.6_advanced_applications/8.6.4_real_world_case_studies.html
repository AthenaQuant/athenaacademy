Part 2: Case Study #1 - GameStop Squeeze (January 2021)

The Epic Gamma Squeeze

Background: GME was a heavily shorted ($19 stock, 140% short interest) brick-and-mortar video game retailer facing structural decline.

January 11-12, 2021 (Week 1)

Price: GME $19.94

Options Intelligence:

  • Call volume spikes to 890,000 contracts (10x normal)
  • Aggressive buying of $20, $25, $30 strikes
  • Put/Call ratio: 0.22 (extremely bullish)
  • OI building in $40, $50+ strikes (implies major move expected)

‚ö†Ô∏è Signal: Someone knows something. Institutional or smart retail positioning for squeeze.

January 13-15 (Wed-Fri)

Price: GME $31.40 (+57%)

Gamma Mechanics Activate:

  • Dealers short gamma must hedge by buying stock
  • Stock rises ‚Üí dealers buy more ‚Üí stock rises more
  • IV explodes from 120% to 300%

January 25-28 (The Squeeze)

Peak Price: GME $483 (+2,400% from Jan 11!)

What Happened:

  • Gamma squeeze + short squeeze = parabolic move
  • Market makers lost control of hedging
  • Retail FOMO piled in
  • Trading halts, broker restrictions implemented
Key Lessons:
  • Lead Time: 10-14 days advance warning from options
  • Signal Quality: 10x normal call volume impossible to miss
  • P/C Extreme: 0.22 ratio = pure speculation/positioning

from dataclasses import dataclass
from datetime import datetime
from typing import List, Dict
import numpy as np

@dataclass
class MarketSignal:
    """Track daily market signals"""
    date: datetime
    spy_pc_ratio: float
    spy_gex: float
    vix_level: float
    vix_call_oi: float
    cboe_skew: float
    signal_count: int
    threat_level: str

def daily_market_scan() -> MarketSignal:
    """
    Daily options intelligence scan

    Returns MarketSignal with all key metrics
    """
    # Fetch current data (pseudo-code)
    spy_pc = get_spy_put_call_ratio()
    spy_gex = calculate_spy_gex()
    vix = get_vix_level()
    vix_calls = get_vix_call_oi()
    skew = get_cboe_skew()

    # Count active signals
    signals = 0
    if spy_pc > 1.2:
        signals += 1
    if spy_gex < 0:
        signals += 1
    if vix_calls / get_vix_put_oi() > 1.5:
        signals += 1
    if skew > 145:
        signals += 1

    # Determine threat level
    if signals >= 3:
        threat = "HIGH - Go Defensive"
    elif signals == 2:
        threat = "ELEVATED - Monitor Closely"
    elif signals == 1:
        threat = "MODERATE - Normal Caution"
    else:
        threat = "LOW - Normal Conditions"

    return MarketSignal(
        date=datetime.now(),
        spy_pc_ratio=spy_pc,
        spy_gex=spy_gex,
        vix_level=vix,
        vix_call_oi=vix_calls,
        cboe_skew=skew,
        signal_count=signals,
        threat_level=threat
    )

def scan_unusual_single_stock_activity(
    tickers: List[str],
    call_vol_threshold: float = 3.0,
    pc_threshold: float = 0.3
) -> Dict[str, dict]:
    """
    Scan for unusual single-stock options activity

    Args:
        tickers: List of tickers to scan
        call_vol_threshold: Call volume vs 20-day avg threshold
        pc_threshold: P/C ratio threshold (bullish if below)

    Returns:
        Dictionary of tickers with unusual activity
    """
    unusual = {}

    for ticker in tickers:
        call_vol = get_call_volume(ticker)
        avg_vol = get_avg_call_volume_20d(ticker)
        pc_ratio = get_put_call_ratio(ticker)

        # Check for unusual activity
        if call_vol / avg_vol >= call_vol_threshold and pc_ratio <= pc_threshold:
            unusual[ticker] = {
                'call_volume': call_vol,
                'volume_ratio': call_vol / avg_vol,
                'pc_ratio': pc_ratio,
                'signal': 'BULLISH - Unusual Call Activity'
            }

    return unusual
