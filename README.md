# Quantitative Trading Course

Welcome to the comprehensive 29-module quantitative trading course. This repository contains interactive modules designed to teach advanced quantitative trading concepts, mathematical foundations, and practical implementation techniques.

## Course Overview

This course covers everything from fundamental price action analysis to advanced risk metrics and portfolio optimization. Each module includes theoretical concepts, mathematical foundations, and practical Python implementations for real-world trading applications.

## Current Module Structure

The course is organized into the following modules:

### Module 1: Price Action Analysis
Foundation of technical analysis and price movement patterns

### Module 2: Linear Algebra
Mathematical foundations for portfolio theory and factor models

### Module 3: Calculus
Derivatives, optimization, and continuous-time modeling

### Module 4: Volume Analysis
Understanding market microstructure and volume dynamics

### Module 5: Statistics
Probability theory, distributions, and statistical inference for trading

### Module 6: Optimization
Portfolio optimization, constraint programming, and algorithmic solutions

### Module 7: Risk Metrics
VaR, CVaR, drawdown analysis, and risk-adjusted performance measures
